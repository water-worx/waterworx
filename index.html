First Time Setup — Reset All Scripts Once!
After rezzing your Fountain, reset all scripts once to initialize the system properly:
  1. Right-click the Fountain object and select Edit
  2. In your SL viewer menu bar, go to Build → Scripts → Reset Scripts
  3. Wait a few seconds for all scripts to initialize — done! ✅

You only need to do this once after first rez. After that, the system manages itself.

  1. Open your Second Life Inventory

Turning Your Fountain ON / OFF

✅ To Turn ON
  1. Click the Panel
  2. Click 🌍 GLOBAL (or a specific ⛲)
  3. Click ⛲ POWER

❌ To Turn OFF

Section 12

Managing Multiple Fountains

ONE Panel can control up to 8 fountains simultaneously — one of the most powerful features!

📡 How It Works

  • Fountains broadcast their presence on a radio channel automatically
  • The Panel listens on the same channel
  • Every 5 seconds, the Panel sends a "ROLLCALL" — fountains respond with "I'm here!"
  • Fountains that don't respond for 6 seconds are automatically removed
📡
It's all automatic! Just ensure the Panel and fountains use the same channel number (default: -31).

🔨 Setup Steps

1

Build your first fountain

Follow the Quick Start (Section 2)

2

Build additional fountains

Rez a new Fountain Box, click once to grant permission — 5 pipes auto-rez! Use the Panel's 🔷 SHAPES menu to pick a formation. Each gets a unique name automatically.

3

Verify connection

Click the Panel — HOME screen shows all fountains. That's it!

🌍 GLOBAL vs ⛲ Individual

🌍 GLOBAL — All At Once
  • Turn all ON/OFF together
  • Same color across all
  • Synchronized shows
  • "One click controls everything!"
⛲ Individual — Just One
  • Different colors per fountain
  • Adjust one fountain's settings
  • Access 🔷 SHAPES & 🔧 RENAME
  • Run a show on one only

🔧 Renaming Fountains

  1. Click Panel → click a ⛲ fountain
  2. Click 🔧 RENAME
  3. Type a new name (up to 30 characters)
  4. Press Enter — done!

Names are truncated to 8 characters on buttons due to SL dialog limits.

📻 Using Different Channels

Create independent groups of fountains by assigning different channels:

Channel -31 (Panel A)
  • ⛲ Fountain 1
  • ⛲ Fountain 2
  • ⛲ Fountain 3
Channel -32 (Panel B)
  • ⛲ Fountain 4
  • ⛲ Fountain 5

To set/change a channel: Touch the object for the first time — you'll be prompted to enter a channel number. Already set? Touch the Panel, go to 🛠 ADMIN → 📡 CHANNEL to change it.

⚠️
Panel and its fountains must use the same channel number! The channel persists across restarts — set each object once.

Advanced Configuration

For power users — customize defaults by editing notecards directly.

� Channel Setup (First Touch)

The communication channel between the Panel and fountains is configured automatically on first touch — no notecard editing needed!

  1. Rez the Panel (or Fountain). Touch it for the first time.
  2. A text-box dialog appears asking for a channel number.
  3. Enter any number (e.g. -31) and press Enter.
  4. The channel is saved permanently in the object even through resets and sim restarts.
💡
Every Panel and Fountain you want to work together must use the same channel. Set each one to the same number on first touch.

Troubleshooting

🔧
Universal Fix (solves 90% of issues): Click Panel → 🛠 ADMIN → ♻️ RESET ALL. Everything returns to defaults!

Check: Is the fountain ON? Is HEIGHT above 0? Was the Fountain Box the root prim?

  1. Go to ⚙️ SETTINGS → 📊 HEIGHT → type 5
  2. Turn fountain ON via ⛲ POWER
  3. Wait 5 seconds
  4. Still not working? Unlink and re-link with Fountain Box selected last.
  1. Right-click Panel → Edit
  2. Build menu → Reset Scripts
  3. Wait 10 seconds, then click panel again

Check: Are both using the same channel? Are they in the same region/sim?

  1. Touch the Panel → go to 🛠 ADMIN → 📡 CHANNEL and note the channel number shown
  2. Touch the Fountain → if it was never configured, a text box will appear — enter the same channel number
  3. Reset both objects' scripts
  4. Wait 5 seconds (one heartbeat cycle)
  5. Click Panel — fountain should appear
  1. Click 🟡 LIGHTS → try 🎭 THEMES → pick any theme
  2. Wait 2 seconds
  3. Still wrong? Go to ⚙️ SETTINGS → enable 🌟 GLOW
  1. Ensure both use the same channel number
  2. 🛠 ADMIN → ♻️ RESET ALL
  3. Turn all ON via 🌍 GLOBAL → ⛲ POWER
  1. GLOBAL → ⛲ POWER (turn OFF)
  2. ⛲ POWER again (turn ON)
  3. Display should now show 0060
  1. Right-click Panel → Edit
  2. Check for small sphere on top
  3. If present: Reset Scripts
  4. If missing: Panel may need rebuild

Check: Is the fountain ON? Does it have ✓ (ANIMATE module)?

  1. Turn fountain ON first
  2. Then go to 🎭 SHOWS → pick a show
  3. If no ✓, the ANIMATE module isn't linked — check fountain contents

Just reset! Click 🛠 ADMIN → ♻️ RESET ALL. This fixes almost everything — all settings return to defaults and the fountain keeps working.

Cause: Pipes may have been manually tilted or the fountain was rotated after setup.

  1. Type /wx:straighten in local chat
